# Phase 2 — Data Wrangling & Analysis

**Goal:** Load, clean, transform, and explore data. This is ~70% of real ML work.
**Time:** 2–3 weeks.

---

## 🎯 Outcomes
You can take a messy real-world dataset and turn it into clean, analyzed, visualized insight.

## ✅ Checklist

### NumPy
- [ ] Arrays, dtypes, shapes, reshaping
- [ ] Indexing, slicing, boolean masks
- [ ] Broadcasting (key concept)
- [ ] Vectorized ops vs Python loops (speed)

### Pandas
- [ ] `Series` & `DataFrame`
- [ ] Reading CSV/JSON/Parquet/SQL
- [ ] Selecting: `loc`, `iloc`, boolean filtering
- [ ] Handling missing data (`fillna`, `dropna`)
- [ ] `groupby`, aggregation, pivot tables
- [ ] Merging/joining DataFrames
- [ ] `apply`, `map`, vectorized string ops
- [ ] Dates & time series basics

### Exploratory Data Analysis (EDA)
- [ ] Summary statistics, distributions
- [ ] Detecting outliers & data quality issues
- [ ] Correlation analysis
- [ ] Feature understanding (categorical vs numerical)

### Visualization
- [ ] Matplotlib (line, scatter, hist, bar)
- [ ] Seaborn (heatmaps, pairplots, distributions)
- [ ] When to use which chart

- **EDA/Viz** — [Kaggle: Data Visualization](https://www.kaggle.com/learn/data-visualization)
- **Book** — *Python for Data Analysis* by Wes McKinney (pandas' creator)

## 🛠️ Phase project
**End-to-end EDA on a real dataset.** Pick something from [Kaggle Datasets](https://www.kaggle.com/datasets) you care about (e.g. your domain). Clean it, find 5 insights, make 5 polished charts, write up findings in a notebook with markdown commentary.
