# 📚 Master Resource List

The best resources, curated. You don't need all of these — pick one per topic and go deep. **Free** unless noted.

## 🎓 Flagship courses
- [Andrew Ng — Machine Learning Specialization](https://www.coursera.org/specializations/machine-learning-introduction) — best ML intro
- [fast.ai — Practical Deep Learning](https://course.fast.ai/) — best top-down DL
- [Karpathy — Neural Networks: Zero to Hero](https://www.youtube.com/playlist?list=PLAqhIrjkxbuWI23v9cThsA9GvCAUhRvKZ) — build GPT from scratch
- [DeepLearning.AI — Deep Learning Specialization](https://www.coursera.org/specializations/deep-learning)
- [Hugging Face — NLP & LLM Courses](https://huggingface.co/learn)
- [Made With ML](https://madewithml.com/) — MLOps, production
- [Full Stack Deep Learning](https://fullstackdeeplearning.com/)
- [DeepLearning.AI — Short Courses](https://www.deeplearning.ai/short-courses/) — 1hr each on RAG, agents, LLMs

## 📖 Books
- *Hands-On ML with Scikit-Learn, Keras & TensorFlow* — Géron **(the practical bible)**
- *Designing Machine Learning Systems* — Chip Huyen **(production)**
- *AI Engineering* — Chip Huyen (2025) — LLM systems
- *Mathematics for Machine Learning* — [free PDF](https://mml-book.github.io/)
- *Dive into Deep Learning* — [d2l.ai](https://d2l.ai/) (free, code-first)
- *Machine Learning Interviews* — Chip Huyen — [free](https://huyenchip.com/ml-interviews-book/)
- *The Hundred-Page Machine Learning Book* — Burkov

## 📺 YouTube channels
